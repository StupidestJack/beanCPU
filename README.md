# beanCPU
## 基於C#的16-bit CPU模擬器

### 專案介紹

beanCPU 是一個輕量級、基於堆疊與暫存器混合邏輯的模擬器專案。<br>
本專案包含了一個自定義的 CPU 指令集 (ISA)、<br>
一個支援語法糖的組譯器 (Assembler，<br>
以及一個具備點陣字體顯示功能的 模擬器 (Emulator)。

### 技術規格
* 字長 (Word Size): 16-bit
* 定址空間: 64 KB (0x0000 - 0xFFFF)
* 暫存器: 8 個通用暫存器 (R0-R7)
* 特殊指標: 程式計數器 (PC)、堆疊指標 (SP)
* 編碼格式: 大端序 (Big Endian)
* 開發語言: C# (.NET / WinForms)

### 注意事項
* 此專案使用WinForms，因此僅可在Windows下執行。
* 此架構並不適合用於開發正式專案
